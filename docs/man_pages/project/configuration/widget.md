# ns widget

### Description

Interactively adds a new iOS/Android widget based on a predefined template.

### Commands

Usage | Synopsis
------|-------
General | `$ ns widget <Platform>`

<% if(isHtml) { %>

### Command Limitations

* You can run `$ ns widget ios` only on macOS systems.
