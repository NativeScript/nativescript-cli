publish
==========

Usage | Synopsis
---|---
General | `$ tns publish ios`

Uploads project to an application store.

### Command Limitations

* You can run `$ tns publish` only on OS X systems.
* Currently, you can run `$ tns publish` to publish only iOS applications.
